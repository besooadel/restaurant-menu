# 🍽️ منيو المطعم الإلكتروني

نظام منيو إلكتروني احترافي متكامل مع لوحة تحكم إدارية.

## 🌟 المميزات

### للعملاء (index.html)
- 🎬 **سلايدر متحرك** للعروض والإعلانات
- 🎥 **دعم الفيديو** لعرض المنتجات
- 🍽️ **تصفية حسب الفئات** مع تأثيرات بصرية
- 🔍 **بحث فوري** عن الوجبات
- 📄 **صفحة تفصيلية** لكل وجبة
- ➕ **إضافات ومكونات** قابلة للتخصيص
- 🛒 **سلة مشتريات** مع رقم الطاولة
- ⚠️ **تعطيل المنتجات** الناضبة بلون باهت
- 📱 **متجاوب** مع الجوال والتابلت

### لوحة التحكم (admin/index.html)
- 🔐 **تسجيل دخول** برقم سري
- 📊 **لوحة إحصائيات** شاملة
- ➕➖ **إضافة وحذف وتعديل** المنتجات
- 📦 **إدارة المخزون** مع تنبيهات
- 📋 **عرض الطلبات** الواردة
- 🎬 **رفع فيديوهات** للمنتجات
- 📱 **واجهة متجاوبة** مع الجوال

## 🚀 التثبيت

### 1. رفع على GitHub Pages
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/restaurant-menu.git
git push -u origin main
```

ثم من إعدادات المستودع → Pages → اختر `main branch`

### 2. إعداد قاعدة البيانات (MySQL)
```bash
mysql -u root -p < database.sql
```

### 3. للاستخدام المحلي
افتح `index.html` مباشرة في المتصفح

## 🔐 بيانات الدخول الافتراضية
- **المستخدم:** admin
- **كلمة السر:** admin2024

⚠️ **مهم:** غير كلمة السر في ملف `admin/admin.js` قبل الرفع
